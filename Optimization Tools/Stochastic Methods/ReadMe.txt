This folder contains four uses of the Monte Carlo method in practical applications. It is used to approximate the value of Pi in MonteCarlo_Pi_Estimate.py. It is used to approximate the value of a definite integral in MonteCarlo_Integrator.py. It is used to assess error propegation through the ideal law equation in MonteCarlo_ErrorPropegation.py. It is used on a product earnings dataset to assess input sensitivities to income in MonteCarlo_ProductEarnings.py.
